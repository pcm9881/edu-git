# git
git이란 분산 버전 제어 시스템입니다. 

## 장점
1. 상태변화를 추적가능
1. 원하는 버전으로 복원가능
1. 프로젝트 무결성 보장
1. 오픈소스로 누구나 무료로 사용가능
